# 功能图片

请在此目录放置功能图片（Feature Graphic）。

## 要求
- 尺寸：**必须** 1024 x 500 像素
- 格式：JPG 或 24位 PNG
- 文件大小：最大 1MB
- 不支持透明背景

## 建议内容
- 应用名称："撸了么"
- 标语："私密的个人健康追踪"或其他突出核心价值的文字
- 应用截图或图标
- 深紫色主题配色

## 如何创建

参考 `FEATURE_GRAPHIC_GUIDE.md` 获取详细说明。

快速方法：
1. 使用 Canva、Figma 等在线工具
2. 创建 1024x500 画布
3. 添加渐变背景（深紫色）
4. 添加应用图标或截图
5. 添加文字（应用名称和标语）
6. 导出为 PNG 或 JPG

## 示例文件名
- `feature_graphic.png`（或 .jpg）
